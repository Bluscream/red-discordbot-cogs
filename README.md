To add these wonderful cogs to your instance, run this command first (`[p]` is your bot prefix):

```
[p]repo add blu https://github.com/Bluscream/red-discordbot-cogs
```

Then install your cog(s) of choice:

```
[p]cog install blu <cog_name>
```

Finally, load your cog(s):

```
[p]load <cog_name>
```

If you don't have an instance, consider using his nice [docker image](https://hub.docker.com/r/phasecorex/red-discordbot)!
